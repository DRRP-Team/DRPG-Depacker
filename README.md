# Doom RPG Resource Depacker

Move here `palettes.bin`, `wtexels.bin` and `mappings.bin` from J2ME Doom RPG or BREW Doom RPG and run:

```sh
node main.js ; bash extract.sh
```

Or use cincrement.cpp
